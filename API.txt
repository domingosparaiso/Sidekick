/upload -> página de upload do firmware
/reboot -> reinicia o dispositivo
/config -> backend de configuração, recebe dados do form
/resources.json -> recursos disponíveis na compilação
/config.json -> configuração atual
/update -> backend de atualização do firmware
/delete -> apaga arquivos da partição SPIFFS
/format -> formata a partição SPIFFS
/fs -> mostra a lista de arquivos no SPIFFS
/fs-upload -> backend de upload de arquivos para o SPIFFS

/button/power -> pressiona o botão power (liga/desliga)
/button/get/power -> lê o valor do botão power
/button/reset/set -> pressiona o botão reset, se não existir relay reset, usa power off/on
/button/get/reset -> lê o valor do botão power
/button/reconfigure/set -> reconfigura a conexão wifi criando um AP
/button/get/reconfigure -> lê o valor do botão reconfigure

/led/power -> lê o valor do LED Power
/led/hdd -> lê o valor do LED HDD

*** Os valores de COMANDO para os relays podem ser:
ON -> Liga o relay e permanece ligado
OFF -> Desliga o relay e permanece desligado
POWER_ON -> Liga o relay, depois de um tempo desliga (usado no relay Power)
POWER_OFF -> Liga o relay, depois de um tempo desliga (usado no relay Power)
POWER_OFF_ON -> Liga o relay, aguarda e desliga, aguarda e liga, aguarda e desliga (usado no relay Power para desligar e ligar o computador quando não existe o relay reset)
RESET -> Liga o relay, depois de um tempo desliga (usado no relay Reset)
*** Todos os tempos são configurados no arquivo "config.h"
/relay/power?cmd=<COMANDO> -> aciona o relay de Power
/relay/reset?cmd=<COMANDO> -> aciona o relay de Reset
/relay/sys1?cmd=<COMANDO> -> aciona o relay do cooler System #1
/relay/sys2?cmd=<COMANDO> -> aciona o relay do cooler System #2
/relay/sys3?cmd=<COMANDO> -> aciona o relay do cooler System #3
/relay/sys4?cmd=<COMANDO> -> aciona o relay do cooler System #4

/rpm/cpu -> Retorna o valor de RPM do cooler da CPU
/rpm/sys1 -> Retorna o valor de RPM do cooler System #1
/rpm/sys2 -> Retorna o valor de RPM do cooler System #2
/rpm/sys3 -> Retorna o valor de RPM do cooler System #3
/rpm/sys4 -> Retorna o valor de RPM do cooler System #4

/temperature/cpu -> Retorna a temperatura do sensor na CPU
/temperature/sys1 -> Retorna a temperatura do sensor System #1
/temperature/sys2 -> Retorna a temperatura do sensor System #2
/temperature/sys3 -> Retorna a temperatura do sensor System #3
/temperature/sys4 -> Retorna a temperatura do sensor System #4
